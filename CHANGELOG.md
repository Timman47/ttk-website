# Changelog

All notable changes to the Toy Time King website will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/).

## [1.1.0] - 2026-01-16

### Changed

- Updated color scheme to green and black to match channel banner
- Reorganized homepage into 3 distinct sections
- Enhanced visual effects with green glow/shadow effects
- Reduced featured videos from 5 to 3 cards
- Replaced "Watch on YouTube" button with embedded latest video player in hero section

### Added

- Featured Videos section with horizontal scrolling carousel
- Scroll buttons for easy video navigation
- 3 actual featured videos from the channel with real thumbnails
- Clickable video cards that link directly to YouTube
- Downloaded YouTube thumbnails (thumbnail1.jpg, thumbnail2.jpg, thumbnail3.jpg)
- JavaScript functionality for carousel navigation

## [1.0.0] - 2026-01-16

### Added

- Initial website launch
- Homepage with hero section featuring channel introduction
- About section with three feature cards highlighting content types:
  - Competitive Battles
  - Strategy Guides
  - Tournament Coverage
- Responsive navigation bar with YouTube channel link
- Pokemon-themed color scheme (Blue, Yellow, Red)
- Responsive design for mobile, tablet, and desktop devices
- Sticky header navigation
- Hover effects and animations on interactive elements
- Footer with copyright information
- README.md with setup instructions
- .gitignore for Python virtual environment

### Design Features

- Modern gradient hero section
- Card-based layout for content features
- Smooth transitions and hover effects
- Mobile-first responsive design
- Accessible color contrast ratios
